พันธะไอออนิก หมายถึง พันธะที่เกิดจากการให้และรับอิเล็กตรอนระหว่างอะตอมของธาตุคู่ร่วมพันธะทำให้เกิดเป็นไอออนบวกและไอออนลบ

     พันธะไอออนิกเป็นพันธะที่เกิดจากแรงยึดเหนี่ยวระหว่างไอออนซึ่งมีประจุไฟฟ้าต่างชนิดกัน เนื่องจากพันธะไอออนิกเกิดจากการให้และรับอิเล็กตรอนระหว่างอะตอมคู่ร่วมพันธะ ดังนั้นธาตุคู่ร่วมพันธะหนึ่งจะต้องเสียอิเล็กตรอนได้ง่าย คือมีพลังงานไอออไนเซชันต่ำ สามารถจะเสียอิเล็กตรอนได้ง่ายโดยไม่ต้องใช้พลังงานมากนัก ได้แก่ ธาตุพวกโลหะ เช่น โลหะอัลคาไลน์(Li,Na,K) และโลหะอัลคาไลน์เอิร์ท(Mg,Ba,Ca) ธาตุเหล่านี้เมื่อให้อิเล็กตรอนไปแล้วส่วนที่เหลือจะครบออกตดส่วนธาตุคู่ร่วมพันธะอีกธาตุหนึ่งจะต้องเป็นธาตุที่มีแนวโน้มรับอิเล็กตรอนได้ดี และเสียอิเล็กตรอนยากคือธาตุที่มีอิเล็กโทรเนกาติวิตีสูงๆนั่นเอง ได้แก่ธาตุพวกอโลหะ เช่น แฮโลเจน ออกซิเจน ไนโตรเจน ธาตุเหล่านี้ต้องการอิเล็กตรอนเพื่อให้ครบออกเตด

การเกิดพันธะไอออนิก

     พันธะไอออนิก เกิดจากธาตุโลหะรวมกับธาตุอโลหะ

พิจารณาตัวอย่างของการเกิดสารประกอบไอออนิก NaCl

     จากโครงสร้างอะตอมของ Na และ Cl โดยที่

11Na มีการจัด e- เป็น 2,8,1 และมี IE1 ต่ำมาก จึงมีแนวโน้มทีจะเสีย e- วงนอกสุด 1 อิเล็กตรอน เกิดเป็นโซเดียมไอออน (Na+) ในขณะที่
17Cl มีการจัด e- เป็น 2,8,7 มี IEและอิเล็กโทรเนกาติวิตีสูง จึงมีแนวโน้มที่จะรับ e- เกิดเป็นคลอไรด์ไอออน(Cl- )
     ซึ่งทั้ง Na+ และCl- มีการจัด e- ครบออกเตด ทำให้อยู่ในภาวะที่เสถียร Na+ 2,8 และ Cl- 2,8,8
          Na + Cl -->[ Na ]+[ Cl ]-
     พันธะที่เกิดจากไอออนที่มีประจุบวกและประจุลบ ดึงดูดกันด้วยแรงดึงดูดระหว่างประจุไฟฟ้า เรียกว่า พันธะไอออนิก นั่นเอง

 

 

 

 

 

 

 

 

 

การพิจารณาความเป็นไอออนิก

โดยทั่วๆไปสารประกอบที่เกิดจากการรวมตัวกันระหว่างโลหะกับอโลหะส่วนใหญ่จะได้สารประกอบที่มีพันธะไอออนิกซึ่งมีสมบัติต่างๆ เช่น นำไฟฟน้าเมื่อหลอมเหลว ละลายน้ำได้เป็นต้น แต่อย่างไรก็ตามสารประกอบไอออนิกไม่ใช่มีแต่สมบัติของความเป็นไอออนิกเท่านั้น แต่มีสมบัติของความเป็นโคเวเลนต์ผสมอยู่ด้วยในทำนองเดียวกันสารประกอบที่มีพันธะโคเวเลนต์ก็มีสมบัติของความเป็นไอออนิกผสมอยู่ด้วยเช่นเดียวกัน ดังจะเห็นได้จากการที่สารประกอบโคเวเลนต์บางชนิด เช่น HCl NH3 CH3COOH ละลายน้ำแล้วได้สารละลายที่นำไฟฟ้าได้ ซึ่งเป็นสมบัติของไอออนิก การพิจารณาว่าสารประกอบชนิดหนึ่งๆจะมีความเป็นไอออนิกมากหรือน้อยพิจารณาได้จากผลต่างระหว่างอิเล็กโทรเนกาติวิตีของอะตอคมคู่ร่วมพันธะ

1. ถ้าอะตอมคู่ร่วมพันธะมีผลต่างของอิเล็กโทรเนกาติวิตีมาก จะมีความเป็นไออนิกมาก(หรือมีความเป็นโคเวเลนต์น้อย)

2. ถ้าอะตอมคู่ร่วมพันธะมีผลต่างของอิเล็กโทรเนกาติวิตีน้อย จะมีความเป็นไออนิกน้อย(หรือมีความเป็นโคเวเลนต์มาก)

กล่าวได้ว่า ยิ่งอะตอมคู่ร่วมพันธะมีอิเล็กโทรเนกาติวิตีต่างกันมากเท่าไร ก็ยิ่งแสดงสมบัติไอออนิกมากเท่านั้น

ตัวอย่างเช่น Na มีอิเล็กโทรเนกาติวิตี = 0.9

O   = 3.5

H   = 2.1

ผลต่างของอิเล็กโทรเนกาติวิตีระหว่าง Na กับO = 3.5-0.9 = 2.6

ผลต่างของอิเล็กโทรเนกาติวิตีระหว่าง H กับO = 3.5-2.1 = 1.4

จะเห็นได้ว่าผลต่างระหว่างอิเล็กโทรเนกาติวิตีของ Na กับ O มากกว่าของ H กับO

ดังนั้นพันธะระหว่าง Na กับO จึงมีสมบัติไอออนิกมากกว่าพันธะของH กับO ซึ่งในที่นี้เสารประกอบระหว่าง Na กับO คือNa2O ส่วนสารประกอบระหว่าง H กับO คือH2O เป็นสารประกอบโคเวเลนต์

โดยทั่วๆไป ถ้าธาตุคู่ร่วมพันธะมีอิเล็กโทรเนกาติวิตีต่างกันมากกว่า 2 จะจัดเป็นพันธะไอออนนิก และถ้าต่างกันน้อยกว่า 2 จะจัดเป็นโคเวเลนต์ ยิ่งต่างกันมากกว่า2 เท่าใดก็ยิ่งเป็นไอออนิกมากเท่านั้น ถ้าอิเล็กโทรเนกาติวิตีต่างกันน้อยเท่าไดก็ยิ่งเป็นโคเวเลนต์มากเท่านั้น ถ้าอิเล็กโทรเนกาติวิตีเท่ากันจะถือว่าเป็นโคเวเลนต์ 100%

    1. พันธะระหว่าง Na กับF ในNaF

      2. Na มีอิเล็กโทรเนกาติวิตี = 0.9

      F มีอิเล็กโทรเนกาติวิตี = 4.0

      ผลต่างของอิเล็กโทรเนกาติวิตี = 4.0-0.9 = 3.1

      อิเล็กโทรเนกาติวิตีต่างกันมากกว่า 2 แสดงว่าเป็นพันธะไอออนิก หรือ NaF เป็นสารประกอบไอออนิก

    2. พันธะระหว่าง Na กับCl ในNaCl

      3. Na มีอิเล็กโทรเนกาติวิตี = 0.9

      F มีอิเล็กโทรเนกาติวิตี = 3.0

      ผลต่างของอิเล็กโทรเนกาติวิตี = 3.0-0.9 = 2.1

      อิเล็กโทรเนกาติวิตีต่างกันมากกว่า 2 แสดงว่าเป็นพันธะไอออนิก หรือ NaF เป็นสารประกอบไอออนิก

    3. พันธะระหว่าง H กับS ใน H2S

      4. H มีอิเล็กโทรเนกาติวิตี = 2.1

      S มีอิเล็กโทรเนกาติวิตี = 2.6

      ผลต่างของอิเล็กโทรเนกาติวิตี = 2.6-2.1 = 0.5

      อิเล็กโทรเนกาติวิตีต่างกันน้อยกว่า 2 แสดงว่าเป็นพันธะโคเวเลนต์ หรือ H2S เป็นสารประกอบโคเวเลนต์

    4. พันธะระหว่าง C กับO ในCO2
      5.

C มีอิเล็กโทรเนกาติวิตี = 2.5

O มีอิเล็กโทรเนกาติวิตี = 3.5

ผลต่างของอิเล็กโทรเนกาติวิตี = 3.5-2.5=1

แสดงว่า CO2 เป็นสารประกอบโคเวเลนต์หรือมีพันธะโคเวเลนต์

เปรียบเทียบสมบัติไอออนิกของสารประกอบทั้ง 4 ชนิด จากค่าอิเลกโทรเนกาติวิตี จะได้ดังนี้

ผลต่างของอิเลกโทรเนกาติวิตี Na-F > Na-Cl > C-O > H-S

เพราะฉะนั้นความเป็นไอออนิก Na-F > Na-Cl > C-O > H-S

ความเป็นโคเวเลนต์ Na-F < Na-Cl < C-O < H-S
