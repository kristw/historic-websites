พลังงานของการเกิดสารประกอบไอออนิก

     เมื่อโลหะทำปฏิกิริยากับอโลหะเกิดเป็นสารประกอบไอออนิก จะมีพลังงานเข้ามาเกี่ยวข้องด้วยเสมอ ซึ่งอาจจะเป็นพลังงานประเภทดูดหรือคายความร้อนได้ เรียกว่าพลังงานของการเกิดสารประกอบไอออนิก

     เช่น เมื่อเผาโลหะโซเดียมให้ร้อน แล้วผ่านก๊าซคลอรีนเข้าไปทำปฏิกิริยา จะได้เป็นสารประกอบไอออนิก NaCl ซึ่งเป็นของแข็ง



เขียนเป็นสมการแสดงการเกิดปฏิกิริยาได้ดังนี้

Na(s) + 1/2 Cl2(g)  --> NaCl(s)


พลังงานของการเกิดสารประกอบไอออนิก NaCl หาได้จากขั้นตอนย่อยๆ คือ


ขั้นที่1 โลหะNa จะดูดพลังงานเข้าไปเพื่อสลายพันธะโลหะและทำให้เปลี่ยนสภาวะจากของแข็ง กลายเป็นก๊าซ (เกิดการระเหิดนั่นเอง)

Na(s)   --> Na(g) ; DHsub

DHsubคือ พลังงานของการระเหิด (Sublimation energy) ซึ่งหมายถึงพลังงานที่ต้องใช้ในการเปลี่ยนสถานะของสารจากของแข็งให้กลายเป็นไอโดยไม่ต้องผ่านสภาวะของเหลว เป็นพลังงานประเภทดูดความร้อน


ขั้นที่2 โซเดียมอะตอมในสภาวะก๊าซ ; Na(g) จะถูกเปลี่ยนให้เป็นโซเดียมไอออนในภาวะก๊าซ Na+ (g) โดยดูดพลังงานเข้าไปจำนวนหนึ่ง ซึ่งก็คือ การเกิดไอออไนเซชันนั่นเอง

Na(g) --> Na+(g) + e- ; IE1

IE1 คือ พลังงานไอออไนเซชั่นลำดับที่ 1 เป็นพลังงานประเภทดูดความร้อน


ขั้นที่3 โมเลกุล Cl2 จะดูดพลังงานเข้าไปสลายพันธะ Cl-Cl ให้เป็น คลอรีนอะตอม

Cl2(g) --> 2Cl(g) ; DHdis

DHdis คือพลังงานของการแตกตัว (dissociation energy) หรือพลังงานพันธะนั่นเอง เป็นพลังงานประเภทดูดความร้อน


ขั้นที่4 คลอรีนอะตอมในภาวะก๊าซ Cl(g) จะเข้าไปรับอิเล็กตรอนจากโซเดียม กลายเป็นคลอไรด์ไอออนในภาวะก๊าซ Cl(g) และคายพลังงานออกมา

Cl(g) + e- --> Cl-(g) ; Ea

Ea คือพลังงานอิเล็กตรอนอัฟฟินิตี (Electron affinity) หรือสัมพรรคภาพอิเล็กตรอน หมายถึง พลังงานที่ให้ออกมา เมื่ออะตอมของธาตุในภาวะก๊าซรับอิเล็กตรอนกลายเป็นไอออนลบในภาวะก๊าซ ดังนั้น Ea จึงเป็นพลังงานประเภทคายความร้อน


ขั้นที่5 โซเดียมไอออนและคลอไรด์ไอออนในภาวะก๊าซจะมารวมกันกลายเป็นผลึกของโซเดียมคลอไรด์ พร้อมกับคายพลังงานออกมาจำนวนหนึ่ง

Na+(g) + Cl-(g) --> NaCl(s) ; Ec

Ec คือ พลังงานโครงร่างผลึก (Crystal lattice energy) หรือพลังงานแลตทิช ซึ่งหมายถึง พลังงานที่ให้ออกมา เมื่อไอออนที่เป็นก๊าซรวมตัวกันกลายเป็นผลึกของแข็งอิออนิก Ec เป็นพลังงานประเภทคายความร้อน


การคำนวณพลังงานของการเกิดสารประกอบไอออนิก

     เมื่อนำการเปลี่ยนแปลงทั้ง5ขั้นย่อยมารวมกันจะได้พลังงานของการเกิดสารประกอบไอออนิกตามต้องการ

เพื่อให้เห็นลักษณะของการเปลี่ยนแปลงอย่างชัดเจนจึงขอนำพลังงานทั้ง 5 ขั้นมาเขียนรวมกันใหม่ดังนี้

Na(s) --> Na(g) ; Hsub = 109 kJ /mol ............................(1)

Na(g) --> Na+(g) + e- ; IE1 = 494 kJ /mol ......................(2)

Cl2(g) --> 2Cl(g) ; Hdis = 121 kJ /mol .............................(3)

Cl(g) + e- --> Cl-(g) ; Ea = -374 kJ /mol ..........................(4)

Na+(g) + Cl-(g) --> NaCl(s) ; Ec = -787 kJ /mol ..............(5)

      เมื่อนำการเปลี่ยนแปลงทั้ง5ขั้นมารวมกันตามหลักคณิตศาสตร์ จะได้ดังนี้

          (1)+(2)+1/2*(3)+(4)+(5) ; Na(s) + 1/2 Cl2(g) --> NaCl(s)

          DH = 109+494+0.5*121+(-347)+(-787)  = -470.5 kJ /mol

     DH คือพลังงานของการเกิด NaCl(s) นั่นเอง โดยดูจากเครื่องหมาย

          ถ้าเป็น + คือ ดูดความร้อน
          ถ้าเป็น - คือ คายความร้อน

     เช่น ในตัวอย่างนี้ คายความร้อน 470.5 kJ/mol

     โดยทั่วๆไป การเกิดสารประกอบไอออนิกอาจจะเป็นประเภทดูดหรือคายความร้อนก็ได้
