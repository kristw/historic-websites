สมบัติทั่วๆไปของสารประกอบไอออนิก

อาจสรุปสมบัติทั่วๆไปของสารประกอบไอออนิกได้ดังนี้

1.ไม่มีสูตรโมเลกุล สารประกอบไอออนิกมีแต่สูตรอย่างง่าย ไม่มีสูตรโมเลกุล ตัวอย่างเช่น กรณีของ NaCl เมื่อNa ซึ่งมีพลังงานไอออไนเซชันต่ำ เสียอิเล็กตรอนให้กับ Cl ไป เหลือเป็น Na+ 2,8 ซึ่งมีการจัดอิเล็กตรอนเหมือน Ne ในขณะที่ Cl มีอิเล็กโตรเนกาติวีตีสูง จะรับอิเล็กตรอนจาก Na เข้ามา กลายเป็น Cl- 2,8,8 ซึ่งมีการจัดเรียงอิเล็กตรอนเหมือน Ar ดังนั้นในระบบจึงมีทั้งไอออนบวกและไอออนลบ ไอออนที่มีประจุตรงกันข้ามจะดึงดูดกันด้วยแรงดึงดูดไฟฟ้าสถิตระหว่างประจุต่างชนิดกัน คือ Na+ จะดึงดูดกับ Cl และ Cl- จะดึงดูดกับ Na+ ต่อเนื่องกันไป เหมือนกับไม่มีที่สิ้นสุด

 

ลักษณะของการดึงดูดกันระหว่างไอออนที่มีประจุต่างชนิดกันอย่างต่อเนื่องไม่สิ้นสุดนี้ ทำให้สารประกอบที่เกิดขึ้นไม่อยู่ในรูปของโมเลกุลเดี่ยว แต่อยู่ในรูปของโครงร่างผลึกที่มีไอออนบวกและไอออนลบดึงดูดต่อๆกันไป จึงทำให้สารประกอบไอออนิกไม่มีสูตรโมเลกุล เพราะไม่ทราบว่าในแต่ละโมเลกุลมีไอออนลบอย่างละเท่าใด

 

เมื่อพิจารณาจากโครงสร้างของ NaCl จะเห็นได้ว่า Na+ แต่ละตัวจะยึดเหนี่ยวกับ Cl ข้างเคียงรวม6ตัวและในทำนองเดียวกัน Cl- 1ตัวก็จะยึดเหนี่ยวอยู่กับ Na+ ที่อยู่ข้างเคียง6ตัวเช่นเดียวกัน แสดงว่าอัตราส่วนของอะตอมของ Na และ Cl ที่มารวมกันเป็นสารประกอบจะเป็น1ต่อ1 ดังนั้นถึงแม้ว่าจะไม่ทราบสูตรโมเลกุล ไม่ทราบว่าใน 1 โมเลกุลมี Na และ Cl อย่างละกี่อะตอม แต่ก็ทราบว่าจำนวนอะตอมของ Na และ Cl ควรจะเท่ากัน จึงเขียนสูตรอย่างง่ายได้เป็น NaCl

เนื่องจากสารประกอบไอออนิกไม่มีสูตรโมเลกุล จึงไม่เรียกสารประกอบที่ยึดเหนี่ยวกันด้วยพันธะ ไอออนิกว่าโมเลกุลไอออนิก ซึ่งแตกต่างกับกรณีสารประกอบโคเวเลนต์ที่มีลักษณะเป็นโมเลกุลเดี่ยว แยกอิสระจากกัน จึงเรียกสารที่ยึดเหนี่ยวกันด้วยพันธะโคเวเลนต์ว่า โมเลกุลโคเวเลนต์ก็ได้

2.ไม่มีมวลโมเลกุล เนื่องจากไม่มีสูตรโมเลกุล ดังนั้นสารประกอบไอออนิกจึงไม่มีมวลโมเลกุล มีแต่มวลสูตร

คือ คิดจากผลบวกของมวลอะตอมตามสูตรอย่างง่ายที่เขียนนั้น

เช่น CaCl2 มีมวลสูตร = 40+2*35.5 = 111

MgO มีมวลสูตร = 24+16 =40

Na2SO4 มีมวลสูตร = 2*23+32+4*16 = 142

แต่อย่างไรก็ตาม ส่วนใหญ่ยังอนุโลมให้เรียกมวลโมเลกุลแทนมวลสูตรได้

3.มีจุดหลอมเหลวและจุดเดือดสูงมาก เนื่องจากพันธะไอออนิกยึดเหนี่ยวกันด้วยไอออนที่มีประจุไฟฟ้าต่างชนิดกัน ซึ่งเป็นพันธะที่แข็งแรงมาก รวมทั้งไอออนเหล่านี้จะยึดเหนี่ยวซึ่งกันและกันแบบต่อเนื่อง เป็นโครงผลึก การที่จะทำให้เปลี่ยนสถานะจากของแข็งเป็นของเหลวหรือจากของเหลวกลายเป็นไอ จึงต้องใช้พลังงานในการทำลายโครงผลึกที่แข็งแรงมาก จุดเดือดและจุดหลอมเหลวจึงสูง ต่างกับสารประกอบโคเวเลนต์ที่ส่วนมาก มีจุดหลอมเหลวและจุดเดือดต่ำ เพราะมีแรงยึดเหนี่ยวระหว่างโมเลกุลน้อยกว่า

4.สภาวะปกติสารประกอบไอออนิกจะเป็นของแข็ง ไม่นำไฟฟ้า แต่จะนำไฟฟ้าได้เมื่ออยู่ในสภาวะหลอมเหลว หรือเมื่อละลายน้ำ ทั้งนี้เพราะในขณะที่เป็นโครงผลึกของแข็ง ไอออนต่างๆที่มีประจุตรงกันข้ามจะยึดเหนี่ยวซึ่งกันและกัน ทำให้ไม่มีไอออนอิสระที่จะเคลื่อนที่เพื่อนำไฟฟ้า แต่เมื่อหลอมเหลวโครงผลึกจะเสียไป ทำให้ไอออนสามารถเคลื่อนที่ได้ จึงสามารถนำไฟฟ้าได้ ในทำนองเดียวกันเมื่อละลายน้ำจะแตกตัวออกเป็นไอออนจึงสามารถนำไฟฟ้าได้ การที่นำไฟฟ้าได้เมื่อหลอมเหลว จัดว่าเป็นสมบัติเฉพาะตัวของสารประกอบไอออนิก

5.เป็นของแข็งที่มีความหนาแน่นสูง แต่ค่อนข้างเปราะ เช่น NaCl เป็นของแข็งซึ่งถ้าทุบ NaCl จะแตกออกจากกัน ทั้งนี้เพราะ การทุบผลึกเป็นการทำให้ไอออนที่มีประจุเหมือนกัน เข้าไปใกล้กันจนแรงผลักระหว่างประจุของไอออนที่เหมือนกันมากกว่า แรงดึงดูดระหว่างไอออนที่มีประจุต่างกัน เกิดความไม่สมดุลทำให้ผลึกแตกออก

6.มีความดันไอต่ำ เนื่องจากในผลึกไอออนิกมีแรงยึดเหนี่ยวระหว่างไอออนมาก ทำให้ระเหยยาก

7.ละลายได้ดีในตัวทำละลายที่มีขั้ว เช่น น้ำ แต่ไม่ละลายหรือละลายได้น้อยมากในตัวทำละลายที่ไม่มีขั้ว เช่น เบนซีน โทลูอีน คาร์บอนเตตระคลอไรด์

8.โดยทั่วๆไปเกิดปฏิกิริยาได้เร็ว โดยเฉพาะในสารละลายหรือในสภาวะก๊าซ

 

พลังงานของการเกิดสารประกอบไอออนิก

เมื่อโลหะทำปฏิกิริยากับอโลหะเกิดเป็นสารประกอบไอออนิก จะมีพลังงานเข้ามาเกี่ยวข้องด้วยเสมอ ซึ่งอาจจะเป็นพลังงานประเภทดูดหรือคายความร้อนได้ เรียกว่าพลังงานของการเกิดสารประกอบไอออนิก

เช่น เมื่อเผาโลหะโซเดียมให้ร้อน แล้วผ่านก๊าซคลอรีนเข้าไปทำปฏิกิริยา จะได้เป็นสารประกอบ

ไอออนิก NaCl ซึ่งเป็นของแข็ง เขียนเป็นสมการแสดงการเกิดปฏิกิริยาได้ดังนี้

Na(s) + 0.5Cl2(g) NaCl(s)

พลังงานของการเกิดสารประกอบไอออนิก NaCl หาได้จากขั้นตอนย่อยๆ คือ

ขั้นที่1 โลหะNa จะดูดพลังงานเข้าไปเพื่อสลายพันธะโลหะและทำให้เปลี่ยนสภาวะจากของแข็ง กลายเป็นก๊าซ (เกิดการระเหิดนั่นเอง)

Na(s) Na(g) ; Hsub

Hsubคือ พลังงานของการระเหิด (Sublimation energy) ซึ่งหมายถึงพลังงานที่ต้องใช้ในการเปลี่ยนสถานะของสารจากของแข็งให้กลายเป็นไอโดยไม่ต้องผ่านสภาวะของเหลว เป็นพลังงานประเภทดูดความร้อน

ขั้นที่2 โซเดียมอะตอมในสภาวะก๊าซ ; Na(g) จะถูกเปลี่ยนให้เป็นโซเดียมไอออนในภาวะก๊าซ Na+ (g) โดยดูดพลังงานเข้าไปจำนวนหนึ่ง ซึ่งก็คือ การเกิดไอออไนเซชันนั่นเอง

 Na(g) Na+(g) + e- ; IE1

ขั้นที่3 โมเลกุล Cl2 จะดูดพลังงานเข้าไปสลายพันธะ Cl-Cl ให้เป็น คลอรีนอะตอม

Cl2(g) 2Cl(g) ; Hdis

Hdis คือพลังงานของการแตกตัว (dissociation energy) หรือพลังงานพันธะนั่นเอง เป็นพลังงานประเภทดูดความร้อน

ขั้นที่4 คลอรีนอะตอมในภาวะก๊าซ Cl(g) จะเข้าไปรับอิเล็กตรอนจากโซเดียม กลายเป็นคลอไรด์ไอออนในภาวะก๊าซ Cl(g) และคายพลังงานออกมาจำนวนหนึ่ง

 Cl(g) + e- Cl-(g) ; Ea

Ea คือพลังงานอิเล็กตรอนอัฟฟินิตี (Electron affinity) หรือสัมพรรคภาพอิเล็กตรอน หมายถึง พลังงานที่ให้ออกมา เมื่ออะตอมของธาตุในภาวะก๊าซรับอิเล็กตรอนกลายเป็นไอออนลบในภาวะก๊าซ ดังนั้น Ea จึงเป็นพลังงานประเภทคายความร้อน

ขั้นที่5 โซเดียมไอออนและคลอไรด์ไอออนในภาวะก๊าซจะมารวมกันกลายเป็นผลึกของโซเดียมคลอไรด์ พร้อมกับคายพลังงานออกมาจำนวนหนึ่ง

 Na+(g) + Cl-(g) NaCl(s) ; Ec

Ec คือ พลังงานโครงร่างผลึก (Crystal lattice energy) หรือพลังงานแลตทิช ซึ่งหมายถึง พลังงานที่ให้ออกมา เมื่อไอออนที่เป็นก๊าซรวมตัวกันกลายเป็นผลึกของแข็งอิออนิก Ec เป็นพลังงานประเภทคายความร้อน

การเปลี่ยนแปลงขั้นย่อยต่างๆทั้ง5ขั้น สามารถแสดงให้เห็นด้วยรูปต่อไปนี้

 

การคำนวณพลังงานของการเกิดสารประกอบไอออนิก

เมื่อนำการเปลี่ยนแปลงทั้ง5ขั้นย่อยมารวมกันจะได้พลังงานของการเกิดสารประกอบไอออนิกตามต้องการ

เพื่อให้เห็นลักษณะของการเปลี่ยนแปลงอย่างชัดเจนจึงขอนำพลังงานทั้ง5ขั้นมาเขียนรวมกันใหม่ดังนี้

 Na(s) Na(g) ; Hsub = 109 kJ mol-1 (1)

Na(g) Na+(g) + e- ; IE1 = 494 kJ mol-1 (2)

Cl2(g) 2Cl(g) ; Hdis = 121 kJ mol-1 (3)

Cl(g) + e- Cl-(g) ; Ea = 374 kJ mol-1 (4)

Na+(g) + Cl-(g) NaCl(s) ; Ec = 787 kJ mol-1 (5)

เมื่อนำการเปลี่ยนแปลงทั้ง5ขั้นมารวมกันตามหลักคณิตศาสตร์ จะได้ดังนี้

(1)+(2)+0.5*(3)+(4)+(5)

Na(s) + 0.5 Cl2(g) NaCl(s)

H = 109+494+0.5*121+(-347)+(-787)

= -470.5 kJ mol-1 (6)

Hใน(6) คือพลังงานของการเกิด NaCl(s) นั่นเอง
