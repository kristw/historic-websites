2. การอินเตอร์เซกชั่น (Intersection)

นิยาม อินเตอร์เซกชั่นของ A และ B คือเซตที่ประกอบด้วย สมาชิกของ A และ B ทั้ง 2 เซต นั่นคือ

A Ç B = { X Î u | X ÎX | A Î B }

A Ç B A Ç B = Æ

A Ç B = A È B B Ì A ได้ A Ç B = B

EX กำหนดให้

A = { 1,2,3,4,5,6,7 }
B = { 1,3,5,7,9 }
C = { 3,4,5,7 }

1. (A Ç B) Ç C = { 3,5,7}

2. A Ç ( B Ç C ) = { 3,5,7}

3. A Ç ( B È C ) = { 1,3,4,5,7}

4. (A Ç B) È (A Ç C) = { 1,3,4,5,7}

5. A È ( B Ç C ) = { 1,2,3,4,5,6,7}

6. (A È B) Ç (A È C ) = { 1,2,3,4,5,6,7}


EX ให้ A = { X Î I | X < 9 } B = { X Î I | X ³ -2 } C = { X Î I | 0 < X £ 9}

A = { 8,7,6,  } 1. A Ç B = { -2, -1,0,  , 8}

B = { -2, -1,0,  } 2. A Ç C = { 1,2,3,  , 8 }

C = { 1,2,3,  ,9 } 3. B Ç C = { 1,2,3,  ,9 }


สมบัติของการอินเตอร์เซกชั่น

1. การสลับที่

A Ç B = B Ç A

2. การเปลี่ยนกลุ่ม

(A Ç B) Ç C = A Ç ( B Ç C )

3. การแจกแจง

A Ç ( B È C ) = (A Ç B) È (A Ç C)

A È ( B Ç C ) = (A È B) Ç ( A È C)

4. A Ç B = B เมื่อ B Ì A

5 . A Ç Æ = Æ

6. A Ç universe = A

7. A Ç A = A

8. ถ้า A Ì C และ B Ì C แล้ว A Ç B Ì C

9. A Ç B Ì A และ A Ç B Ì B


3. คอมพลิเมนต์ (Complement)

กำหนด U = { 1,2,3,  , 10} และ A = { 1,3,5,7,9}

เรียก{ 2,4,6,8,10} ว่า คอมพลิเมนต์ของเซต A

นิยาม กำหนดให้ A Ì universe

คอมพลิเมนต์ของเซต A คือ เซตที่ประกอบด้วยสมาชิกซึ่งเป็นสมาชิกของ universe แต่ไม่เป็นสมาชิกของเซต A

สัญลักษณ์ที่ใช้ A' หรือ c[A]

หรือ A' = { x Î universe | X Ï A }

EX กำหนดให้ u = { 10, -9 , -8,  ,8,9,10}

A = { 1,2,3,  ,10}

B = { 2,4,6,8,10}

C = { -1,-3,-5,-7,-9,-10}

จงหา A', B', C'

A'= { 0,-1,-2,  ,-10}

B'= { -10,-9,-8,  ,0,1,3,5,7,9}

C'= { -8,-6,-4,-2,0,1,2,  ,10}

สมบัติของคอมพลิเมนต์

ถ้า A และ B เป็นสับเซตของเอกภพสัมพัทธ์แล้ว

1. (A')'= A
2. Æ' = u
3. universe' = Æ
4. A Ç A' = Æ
5. A È A'= u
6. ( A Ç B)'= A' È B'
7. ( A Ç B)'= A' È B'
8. A Ì B ก็ต่อเมื่อ B' Ì A'
9. A Ç B = Æ ก็ต่อเมื่อ A Ì B'

4. ผลต่าง (Difference)

นิยาม กำหนดให้ A,B เป็นเซตชนิดใดๆ ผลต่างระหว่างเซต A และ B คือเซตที่ประกอบด้วยสมาชิกซึ่งอยู่ในเซต A แต่ไม่อยู่ในเซต B เขียนแทนด้วย A-B นั่นคือ

A-B = { X | X Î A และX Ï B }
A-B เมื่อ B Ì A A-B = Æ ; A = B A-B = Æ ; A Ì B

EX
A = { 1,2,3} B = { 2,4,6} จงหา A-B , B-A

A-B = { 1,3}

B-A = { 4,6}

สมบัติของ Difference

1. A-A = Æ
2. Æ -A = Æ
3. A-Æ = A
4. A' = universe - A
5. A-B ¹ B-A ยกเว้น A = B
6. A-B = A Ç B'
7. A-B = A - (A Ç B)
8. A - (B Ç C) = (A-B) È (A-C)
9. A - (B È C) = (A-B) Ç (A-C)
10. ( A-B) - C = A - (B È C)
11. A - (B-C) = (A-B) È (A Ç C)
12. A Ç (B-C) = (A Ç B) - (A Ç C)
13. A Ç (B-C) = (A Ç B) - C
14. A- (B-C) = (B'-A') È (A Ç C)
15. A-B = A ก็ต่อเมื่อA Ç B = Æ
16. A-B = Æ ก็ต่อเมื่อ A Ì B
17. A-B = B'- A'
18. (A-B) - C = (A-C) - B

 

EX พิสูจน์ (A Ç B Ç C) È (A' Ç B Ç C) È (B' È C') = u

u = [A È A'Ç B Ç C] È (B Ç C)'

= [u Ç (B Ç C)] È (B Ç C)'

= (B Ç C) È (B Ç C)'

= u

5. เอกซ์คลูซีฟยูเนียน (Exclusive Union)

นิยาม กำหนดให้ และ เป็น สับเซตของเอกภพสัมพัทธ์

เอกคลูซีฟยูเนียนของ และ คือ เซต (A-B) È (B-A)

กล่าวคือ A È B = (A-B) È (B-A) นั่นคือ

A È B = { X Î universe | X Î A หรือ X Î B และ X Ï A Ç B}

A È B Ì A È B และ A È B ¹ A È B A È B = A È B เมื่อ A Ç B = Æ

A È B = Æ เมื่อ A = B A È B = B-A เมื่อ A Ì B
