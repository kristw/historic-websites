1. ยูเนียน ( Union )

นิยาม กำหนดให้ A , B เป็นเซตใดๆ ยูเนียนของเซต A , B คือ เซตที่ประกอบด้วยสมาชิกซึ่งอยู่ในเซต A หรือ เซต B หรืออยู่ทั้งในเซต A และ เซต B เขียนแทนด้วย A U B นั่นคือ

A U B = { x | x ฮA หรือ x ฮB}

Ex. กำหนด A = { 1,2,3,4,5} B = { 3,4,6} C = { 5,7,8}

จงหา 1. A U B 2. B U A 3. A U C 4. C U A

5. B U C 6. C U B 7. (A U B) U C 8. A U (B U C)

  1. A U B = { 1,2,3,4,5,6}
  2. B U A = { 1,2,3,4,5,6}
  3. A U C = { 1,2,3,4,5,7,9}
  4. C U A = { 1,2,3,4,5,7,9}
  5. B U C = { 3,4,5,6,7,9}
  6. C U B = { 3,4,5,6,7,9}
  7. (A U B) U C = { 1,2,3,4,5,6,7,9}
  8. A U (B U C) = { 1,2,3,4,5,6,7,9}

คุณสมบัติของยูเนียน
  1. คุณสมบัติการสลับที่

    2. ถ้า A , B เป็นเซตใดๆ แล้ว A U B = B U A

  2. คุณสมบัติการเปลี่ยนกลุ่ม

    3. ถ้า A , B, C เป็นเซตใดๆ แล้ว (A U B) U C = A U (B U C)

  3. ให้ A , B เป็นเซตใดๆ จะได้ว่า A U B = B เมื่อ A Ì B
  4. ถ้า A เป็นเซตใดๆแล้ว A U Æ = A
  5. ถ้า A เป็นเซตใดๆของ u แล้ว A U u = u
  6. ถ้า A เป็นเซตใดๆแล้ว A U A = A
  7. ถ้า A และ B เป็นเซตใดๆ แล้ว A Ì A U B และ B Ì A U B
  8. ถ้า B Ì A และ C Ì A แล้ว B U C Ì A
  9. A U B = Æ ก็ต่อเมื่อ A = Æ และ B = Æ

 

 

หลักการหาจำนวนเซต

1. ถ้า B เป็นเซตจำกัด และ A Ì B แล้ว จำนวนของเซต x โดยที่ A Ì x และ x Ì B จะเท่ากับ 2m-n เซต เมื่อ m = จำนวนสมาชิกของ B

n = จำนวนสมาชิกของ A

Ex. กำหนด กำหนด A = { a} B = { a,b,c,d,e} จงหาจำนวนของเซต A โดยที่ A Ë x และ x Ì B

A Ë x Ì B

2m-n = 25-1 = 16 เซต

2. ถ้า B เป็นเซตจำกัด และ A Ì B โดยที่ A มีสมาชิกเพียงตัวเดียวแล้ว จำนวนของเซต x โดยที่ A Ë x และ x Ì B จะเท่ากับ 2m-n เซต เมื่อ m = จำนวนสมาชิกของ B

3. ถ้า B เป็นเซตจำกัด และ A Ì B แล้ว จำนวนของเซต x โดยที่ A Ë x และ x Ì B = (จำนวนเซต x โดยที่ x Ì B ) - (จำนวนเซต x โดยที่ x Ì A และ x Ì B ) = 2m - 2m-n เมื่อ m = จำนวนสมาชิกของ B

n = จำนวนสมาชิกของ A
