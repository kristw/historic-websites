คุณสมบัติของยูเนียน

 1. คุณสมบัติการสลับที่ ถ้า A , B เป็นเซตใดๆ แล้ว A U B = B U A
2. คุณสมบัติการเปลี่ยนกลุ่ม ถ้า A , B, C เป็นเซตใดๆ แล้ว (A U B) U C = A U (B U C)
3. ให้ A , B เป็นเซตใดๆ จะได้ว่า A U B = B เมื่อ A Ì B
4. ถ้า A เป็นเซตใดๆแล้ว A U Æ = A
5. ถ้า A เป็นเซตใดๆของ u แล้ว A U u = u
6. ถ้า A เป็นเซตใดๆแล้ว A U A = A
7. ถ้า A และ B เป็นเซตใดๆ แล้ว A Ì A U B และ B Ì A U B
8. ถ้า B Ì A และ C Ì A แล้ว B U C Ì A
9. A U B = Æ ก็ต่อเมื่อ A = Æ และ B = Æ


หลักการหาจำนวนเซต

 เมื่อ m = จำนวนสมาชิกของ B       n = จำนวนสมาชิกของ A

1. ถ้า B เป็นเซตจำกัด และ A Ì B แล้ว จำนวนของเซต x โดยที่ A Ì x และ x Ì B
         จะเท่ากับ 2m-n เซต

2. ถ้า B เป็นเซตจำกัด และ A Ì B โดยที่ A มีสมาชิกเพียงตัวเดียวแล้ว จำนวนของเซต x โดยที่ A Ë x และ x Ì B
          จะเท่ากับ 2m-n เซต

3. ถ้า B เป็นเซตจำกัด และ A Ì B แล้ว จำนวนของเซต x โดยที่ A Ë x และ x Ì B
           = (จำนวนเซต x โดยที่ x Ì B ) - (จำนวนเซต x โดยที่ A Ì x และ x Ì B )
           = 2m - 2m-n
