ความหมายของเซต

    เซต เป็นอนิยาม (Undefined term) คำว่า เซต ใช้แทนสิ่งที่เป็นกลุ่มของคน, สัตว์ หรือสิ่งของ ซึ่งมีความหมายเดียวกับคำหลายคำในภาษาไทย เช่น กลุ่ม, หมู่, ฝูง, ชุด ฯลฯ

    สิ่งที่อยู่ในเซต เรียกว่า สมาชิก (elements) ของเซต ใช้สัญลักษณ์ Î แทนคำว่า เป็นสมาชิกของ และใช้สัญลักษณ์ Ï แทนคำว่า ไม่เป็นสมาชิกของ
    เช่น  กำหนดให้ A = { ช้าง, ม้า, วัว, ควาย} จะได้คำว่า ช้าง Î A, ม้า Î A, สุกร Ï A, ไก่ Ï A

    การเขียนชื่อเซตโดยทั่วไปนิยมใช้อักษรตัวพิมพ์ใหญ่ เช่น A, B, C ฯลฯ เรียกชื่อเซต และใช้อักษรตัวพิมพ์เล็ก เช่น a, b, c ฯลฯ เขียนแทนสมาชิก
    เช่น A = {a, e, i, o, u}

วิธีการเขียนเซต มี 2 วิธี

1. วิธีเขียนแบบแจกแจงสมาชิก เป็นวิธีการเขียนเซต โดยเขียนสมาชิกแต่ละตัวของเซตลงในเครื่องหมายวงเล็บปีกกา และใช้เครื่องหมายจุลภาคคั่นระหว่างสมาชิกแต่ละตัว ถ้าสมาชิกมีจำนวนมากให้ใช้เครื่องหมาย  ต่อท้าย

         หมายเหตุ

          1. เซตใดที่มีสมาชิกตัวเดียวกันปรากฏมากกว่า 1 ครั้ง ให้ถือว่ามีสมาชิกเพียงตัวเดียว เช่น { 1, 2, 1} = { 1, 2}
          2. สมาชิกภายในเซตสามารถสลับที่กันได้ โดยไม่ถือตำแหน่ง เช่น { 1, 2, 3} = { 3, 2, 1} = { 2, 1, 3}
          3. สมาชิกในเซต อาจเป็นเซตอีกก็ได้ เช่น A = { a, { b} , { c, d} } จะมีจำนวนสมาชิกทั้งหมด 3 ตัว

          ตัวอย่าง
           1. เซตของสระในภาษาอังกฤษ เขียนเป็น A = { a, e, i, o, u}
            2. เซตของจำนวนคู่ เขียนเป็น B = { , -2, -4, 2, 4, 6, }
            3. เซตของจำนวนนับตั้งแต่ 1 ถึง 100 เขียนเป็น C = { 1, 2, 3, , 100}
            4. เซตของจำนวนเต็ม เขียนเป็น D = { 0, ± 1, ± 2, ± 3, }

2. วิธีแบบบรรยายลักษณะหรือบอกเงื่อนไข ทำโดยใช้วงเล็บปีกกาล้อมรอบตัวแปรที่เป็นตัวแทนของสมาชิกในเซตพร้อมทั้งบอกเงื่อนไขการเป็นสมาชิก โดยใช้เครื่องหมาย  |  หรือ  :  หรือ ; คั่นระหว่างตัวแปรและเงื่อนไข

สัญลักษณ์แทนเซตของจำนวนต่าง ๆ

I   แทนเซตของจำนวนเต็ม
I+   แทนเซตของจำนวนเต็มบวก
I-   แทนเซตของจำนวนเต็มลบ
I 0   แทนเซตของจำนวนศูนย์
Q   แทนเซตของจำนวนตรรกยะ
Q+   แทนเซตของจำนวนตรรกยะที่เป็นบวก
Q-   แทนเซตของจำนวนตรรกยะที่เป็นลบ
R   แทนเซตของจำนวนจริง
R+   แทนเซตของจำนวนจริงบวก
R-   แทนเซตของจำนวนจริงลบ
N   แทนเซตจำนวนนับ
O   แทนเซตของจำนวนคี่
E   แทนเซตของจำนวนคู่
