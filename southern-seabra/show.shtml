     Project Objectives

     1. เพื่อพัฒนาการเรียนรู้ในรายวิชา ส 503 ภูมิศาสตร์กายภาพโดยมีการศึกษานอกเหนือจากตำราเรียนซึ่งเป็นการค้นคว้าความรู้เพิ่มเติมจากภายนอกให้มีความรู้ความเข้าใจอย่างถ่องแท้
     2. เพื่อเป็นการสร้างผลงาน ที่เป็นประโยชน์ต่อผู้ศึกษา โดยนำเสนอในรูปแบบของ Web Site ซึ่งเป็นการนำเสนอที่สามารถปรับปรุงเปลี่ยนแปลง ได้อยู่เสมอ ซึ่งจะทำให้เป็นชิ้นงานที่สามารถก้าวหน้าไปอย่างไม่หยุดยั้ง และทันสมัยอยู่เสมอ
     3. เพื่อเป็นการนำเสนอความรู้และความเข้าใจเกี่ยวกับทะเลและทรัพยากรต่างๆ อีกทั้งแหล่งท่องเที่ยว แก่ผู้ที่เข้ามาเยี่ยมชม โดยอาศัย Internet เป็นสื่อกลางในการนำเสนอ
     4. เพื่อเป็นการส่งเสริมทักษะในการทำงานร่วมกันในกลุ่มอีกทั้งยังเป็นการสร้างความสามัคคีในกลุ่มเพื่อนอีกด้วย

     Before became this site
     ก่อนอื่นก็คงต้องขอบคุณมิส สุกัญญาที่ได้ ระเบิด ประกายความคิดของเราจนออกเป็นงานนี้ได้ กว่าจะออกมาได้ถึงขนาดนี้ก็เหนื่อยเหมือนกัน พูดจริงๆว่ามันก็ไม่ง่ายเท่าไหร่หรอกนะครับ กว่าจะหาเนื้อหาได้ แล้วสามารถผ่านสายตาอันแหลมคมของมิสได้ ก็แทบกระอักเลือดแล้ว แต่แน่นอนครับ ถ้าคนเรามีความพยายาม จะทำอะไรมันก็ย่อมสำเร็จ จริงมั้ยครับ ผมจึงบากบั่นต่อไป ไม่ว่าจะเหมือนต้องเข็นครกขึ้นภูเขาอีกกี่ครั้งก็ตาม แล้วความพยายามของผมมันก็ออกมาเป็นอย่างที่เห็นอยู่นี้ไงครับ ท่านผู้ชม
